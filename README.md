# Student Management System (React)

This project is a frontend-only Student Management System built using React.js.

All CRUD operations are handled using in-memory state without any backend.

## Features

• Display student list (Name, Email, Age)
• Add new student with validation
• Edit student with prefilled form
• Delete student with confirmation dialog
• Simulated loading state
• Export student data to Excel
• Responsive UI using Tailwind CSS

## Technologies Used

React.js (CDN)
Tailwind CSS
SheetJS (Excel export)
JavaScript (ES6)

## Running the Project

1. Download the repository
2. Open index.html in a browser

OR run a simple server:

python -m http.server
